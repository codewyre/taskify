# Sample Todo List Application

Done with:

- SpringBoot (API)
- Solid.js (Web App)
- Keycloak (Authentication)


### Initialize
```sh
# Install dependencies for API and Web Project
make init
```

### Build API & Web Application
```sh
# Start thirdparty tools like reverse proxy, idp and db
make build
```

### Start ThirdParty Services
```sh
# Start thirdparty tools like reverse proxy, idp and db
make start
```
